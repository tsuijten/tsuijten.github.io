{%- comment -%} Browser-language detection (inspired by hack-the-planet.io). Runs synchronously in so there is no content flash. Only included on the default-language homepage (see head.html guard) — visiting any other URL is treated as an explicit choice. Decision flow: 1. Crawlers are never redirected (so every hreflang variant stays indexable on its own URL). 2. A stored choice (localStorage `lang`, written by the switcher) always wins: if it differs from this page and an alternate exists → redirect. 3. Otherwise, if the visitor prefers Dutch → stay; if not → go to English. Alternate URLs are read from the tags, so per-language slugs (/spreker/ <-> /en/speaking/) are handled automatically. {%- endcomment -%}
